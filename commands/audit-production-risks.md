# 프로덕션 리스크 감사 (단계 8.5)

`persona-suite:auditing-production-risks` 스킬을 실행한다. 대상: "$ARGUMENTS"

**호출 범위**: 이 커맨드는 *리스크 감사(8.5)만* 실행한다 — 독립 읽기 전용 서브에이전트가 체크리스트 판정표(위반/통과/해당없음/확인불가/목록외 + `파일:라인` 근거)를 산출하고, 위반은 사용자가 항목별 수용/기각 → 수정 → 재감사 1회로 닫는다.
**다음 단계는 자동 실행하지 않는다** — 감사 후 PR·코드리뷰(LIFECYCLE 9단계, 예: code-review 류)로 사용자가 직접 진행.
